# Twitter-Followers-Project
We have developed algorithms that make some processes on a CSV file that contain big data about some Twitter Users (2 million lines in CSV file and more that 80K accounts), Each row represents a certain userâ€™s id and the id of another user he/she follows.


## Features of Project 

* Reading the `CSV` file.
 
* Remove duplicate data from the file. 

* Make some operation to get any amount of `Top influencers`, even if you wish to include all accounts. 

* Get the number of followers of the given `Id`.

* recommend to this `Id` the common followers between your followers.
